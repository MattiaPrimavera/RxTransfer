# RxTransfer

## Setup
Add the JitPack repository in your build.gradle (top level module):

```
allprojects {
    repositories {
        jcenter()
        maven { url "https://jitpack.io" }
    }
}
```

And add next dependencies in the build.gradle of the module:

```
dependencies {
		compile 'com.github.MattiaPrimavera:RxTransfer:0.11'
}
```
